# fertvida-process-automation
Automação de processos administrativos do ClinicSys com Python e Excel, desenvolvida durante minha jornada de transição para DevOps.

# 🏥 ClinicSys Automation | FertVida Prime - FERTGROUP

Projeto desenvolvido para automatizar processos administrativos realizados no sistema ClinicSys, reduzindo tarefas repetitivas e aumentando a produtividade da equipe.

## 📍 Sobre o projeto

Atualmente trabalho na **FertVida Prime - FERTGROUP**, em Teresina - PI, na área de Gestão de Atendimento.

Durante minha rotina identifiquei diversos processos realizados manualmente, consumindo muito tempo e sujeitos a erros.

Decidi iniciar este projeto para desenvolver uma solução utilizando Python, automatizando essas atividades.

Além de resolver um problema real da empresa, este projeto faz parte da minha jornada de transição para a área de DevOps e Automação.

---

## 🎯 Objetivos

- Automatizar leitura de relatórios do ClinicSys
- Processar arquivos Excel automaticamente
- Preencher planilhas sem intervenção manual
- Reduzir tempo operacional
- Minimizar erros de digitação
- Aprender Python na prática
- Documentar toda a evolução do projeto
