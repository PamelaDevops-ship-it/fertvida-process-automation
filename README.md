# 🏥 ClinicSys Data Exporter | FertVida Prime

Automação de extração de relatórios mensais do ClinicSys para Google Sheets.

## 📍 Sobre

Extraio **manualmente** dados do ClinicSys todo mês (pacientes por médico) e preencho uma planilha.
Este projeto **automatiza** isso com Python + Google Sheets API.

**Tecnologias:** Python | Selenium | Pandas | Google Sheets API | Bash | Azure Cloud Shell
